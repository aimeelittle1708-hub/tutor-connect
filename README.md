# Tutor Connect

Tutor Connect is a web platform that helps students find and book sessions with qualified tutors across various subjects.

## Features
- Tutor profiles and reviews
- Booking and scheduling system
- Secure login and role-based access
- Messaging between students and tutors

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL
- Auth: Firebase Auth
